# 🌐 Prithish Misra — Portfolio Website

This is the **personal portfolio website of [Prithish Misra](https://www.linkedin.com/in/prithish-misra-8531a2341)** — showcasing experience in **Growth, Business Development, AI-driven solutions, and Entrepreneurship**.  
Built entirely with **HTML, CSS, and Vanilla JavaScript**, it features a responsive, elegant layout with **light/dark mode**, project showcases, and a contact form.

---

## 🧩 Project Overview

This portfolio serves as a personal branding and professional showcase platform for:

- Highlighting **academic background** (B.Tech CSE @ SRM KTR ’29)  
- Showcasing **projects** and hackathon submissions (AI, EdTech, ML)  
- Displaying **experience** in business growth, startups, and marketing  
- Providing easy ways to **connect** via LinkedIn, GitHub, and a contact form  

The design is clean, minimal, and fully responsive — adapting seamlessly for mobile, tablet, and desktop.

---

## 🖥️ Features

✅ **Responsive Layout** – Works across all screen sizes  
🌗 **Dark / Light Mode** – Auto-detects system theme and saves user preference  
📁 **Projects Section** – Displays live demos and GitHub links  
💼 **Experience Cards** – Summarizes professional and startup experience  
📬 **Contact Form** – Simple front-end form structure (customizable for backend integration)  
🎨 **Modern UI** – Uses CSS variables for easy theme customization
